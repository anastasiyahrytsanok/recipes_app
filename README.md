# 🍽️ Tasty Recipes

A modern recipe management application built with React, TypeScript, Redux Toolkit, and TanStack Router.

---

## 🚀 Functionality

- 🔐 Authentication (login, token refresh, session restore)
- 📋 View list of recipes (from API + local storage)
- 🔍 Search recipes by name
- 🏷 Filter recipes by tags
- ➕ Create new recipes
- ✏️ Edit your own recipes
- 🗑 Delete recipes (with confirmation modal)
- 📖 View detailed recipe information
- 💾 Local storage support for user-created recipes
- 📱 Responsive UI

---

## 📦 Dependencies

Main libraries used in the project:

- **React** — UI library
- **TypeScript** — static typing
- **Redux Toolkit** — state management
- **RTK Query** — API data fetching and caching
- **TanStack Router** — routing
- **React Redux** — bindings for Redux
- **Webpack** — module bundler

---

## ⚙️ Setup & Run

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm start
```

### 3. Build project

```bash
npm install
```
